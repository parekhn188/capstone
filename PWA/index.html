Features

Select a chord

Select a chord from the drop down menu and receive information about that chord, including name, finger location, and the ability to strum in order to reproduce the sound from that chord

Play a song

Select a list of chords to play, strum to reproduce the sound and press the button to switch chords, allowing for you to play song.

String UI

Use the string UI to practice your finger placement and have the complete guitar playing experience.
